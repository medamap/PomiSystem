# Task 01: X1 色相重視（肌色改善）ロジック設計（実装最小）

## 目的
- 色相重視の重み付けロジックを定義し、pomi-x1.js に最小追加で入れる

## 仕様（提案ロジック）
- RGB -> HSV に変換して「肌色帯」を強調する
- 肌色帯の中心: Hue 25°〜40°（オレンジ寄り）
- 距離計算を「色距離 * 重み」で補正

### 具体式
1) RGB -> HSV（0..360 / 0..1 / 0..1）
2) 肌色距離 d = min( |h - 30|, 360 - |h - 30| )
3) 重み w = 1 - clamp(d / 40, 0, 1)  // 0〜40度以内で最大
4) 補正係数 k = 1 - (w * 0.35)  // 0.65〜1.0（肌色が近いほど距離が縮む）
5) 色距離 = baseDist * k
