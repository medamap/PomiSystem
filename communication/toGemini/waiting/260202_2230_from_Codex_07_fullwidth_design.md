# Task 07: 全幅モードの設計（MSX/X1 共通）

## 目的
- MSX/X1 共通で適用される「全幅モード」の設計を明確化する

## 仕様案
1) UI: トグル名は「全幅モード」
2) 反映対象: Quick 画面全体のコンテナ幅/余白
3) 既定値: 標準（OFF）
4) 保存: localStorage に保存（キー例: quickFullWidth）

## 実装方針
- Quick.razor に bool を追加し、body/root に CSS クラスを付与
- app.css でクラスに応じたレイアウト切替

## 禁止
- 大規模なレイアウト再設計は禁止
- リファクタリングは行わない
