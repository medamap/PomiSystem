# X1 差異整理（MSX Quickとの差分）

## 主要パラメータ差分

### 解像度モード
- X1: X1turbo 320×200 を基準に x2/x4 + Auto + 各種高解像度
- MSX: 256×192 を基準に msx/msx2/msx4 + Auto + 各種高解像度
- X1 では高解像度は「無制限PCGモード自動」

### 色数/パレット
- X1: デジタル8色パレット（固定）
- MSX: TMS9918 16色パレット（固定）
- X1: 「色相重視（肌色改善）」の距離関数オプションあり

### タイル制約
- X1: 8×8 PCGタイル単位（PCG定義数上限）
- MSX: 8×1ライン2色制約（タイル単位というより“ライン制約”）

### タイルリデュース
- X1: タイル類似度による統合・削減あり
  - 類似度しきい値
  - エッジ保護（Sobel）
  - 無制限モードでスキップ可
- MSX: タイル統合なし（ライン2色へ単純化）

### ディザ
- X1: none / bayer2 / bayer4 / floyd / atkinson
- MSX: 上記 + checker / stripe / stripe_strong

### 補正
- X1: 明るさ/コントラスト/彩度 + 色相重視
- MSX: 明るさ/コントラスト/彩度のみ

### 出力
- X1: 画像出力 + PCG定義画像
- MSX: 画像出力 + SC2(BLOAD)出力

## 機種パラメータ化候補（差分吸収点）

- 基準解像度（MSX:256×192 / X1:320×200）
- 固定パレット（MSX 16色 / X1 8色）
- タイル制約タイプ
  - MSX: line2color
  - X1: tile-pcg
- タイル削減有無（X1 only）
- 色距離関数（X1: hueWeight option）
- 高解像度時の制限（X1: unlimited auto）
- 出力形式（MSX: SC2 / X1: PCG image）
