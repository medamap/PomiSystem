# 共通モジュール候補リスト（初回ドラフト）

## 画像入力/前処理
- 画像ロード（DataURL/Blob/URL）
- EXIF/Orientation 補正
- クロップ（中央/比率/位置指定）
- リサイズ（最近傍/バイリニア/バイキュービック）
- 色補正（明るさ/コントラスト/彩度/ガンマ）

## 減色・パレット
- 目標パレット生成（K-means/Median-cut/固定パレット）
- パレットマッピング（最近色 / 距離関数差し替え）
- 2色制約（ライン/ブロック単位の2色化）

## ディザ/ノイズ
- Ordered: チェッカー / Bayer 2×2 / Bayer 4×4
- Error diffusion: Floyd-Steinberg / Atkinson
- パターンディザ: 縦縞 / 横縞 / ドット
- 強度パラメータ適用

## タイル/ブロック処理
- タイル分割（8×8/16×16/可変）
- タイル統計（色頻度/代表色）
- タイル内減色（タイルごとのパレット制約）
- タイルリデュース（同一タイル統合 / 近似統合）

## 出力
- 画像出力（PNG/RAW）
- 機種別バイナリ組み立て（CHR/VRAM/パレット/ネームテーブル）

## UI共通
- プレビュー拡大縮小
- スライダー連動UI
- 自動更新／手動変換
- 変換中UIロック／完了通知
