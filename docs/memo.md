# Memo

## Rule

- カテゴリ毎にセクションを分ける（MSX / X1 / Common など）。
- メモは (section)_(class)_memo.md とする。
  - (section) には MSX / X1 / Common などが入る。
  - (class)
    - onetime ... (section) に対する一時的なメモ。用が済んだら削除する。
    - global0001 ... 残しておいた方が良い情報。増えたら番号を上げて新規追加。

## MSX

- msx_onetime_memo.md
- msx_global0001_memo.md

## X1

- x1_onetime_memo.md
- x1_global0001_memo.md

## Common

- common_onetime_memo.md
